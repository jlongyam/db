# cpsr

[github.com/jlongyam/cpsr](https://github.com/jlongyam/cpsr)

web storage data handler. Using sessionStorage as primary persitance.

Later can be syncronize with larger size storage, localStorage or idb.

## What is this

CPSR meaning

- **C**apture
- **P**review
- **S**tore
- **R**etrive
